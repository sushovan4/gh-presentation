A shape can be modeled as a metric space $(X,d_X)$.

We need an appropriate notion of a distance measure $d_?(X,Y)$ so that


  1. $d_?(X,Y)$ large $\iff$ very different shape
  2. $d_?(X,Y)$ small $\iff$ $X=Y$ upto a class of deformation.

Hausdorff Distance



Nearest neighbor distance Let $A,B\subseteq\mathbb{R}^N$. The directed Hausdorff distance from $A$ to $B$ is defined by $$\overrightarrow{d}_H(A,B)=\sup_{a\in A}\inf_{b\in B}\|a-b\|.$$
For any two compact $A,B\subseteq\mathbb{R}^N$,
  1. $d_H(A,B)\geq0$,
  2. $d_H(A,B)=d_H(B,A)$,
  3. $d_H(A,B)=0\iff A=B$, and
  4. $d_H(A,C)\leq d_H(A,B)+d_H(B,C)$.


$d_H$ is sensitive to Shape + Size + Placement.

Hausdorff under Euclidean Isometry

Let $\mathcal{E}(N)$ denote the class of Eulidean isometries or distance preserving maps $T:\mathbb{R}^N\to\mathbb{R}^N$ i.e. $\|T(a)-T(b)\|=\|a-b\|$.

$$d_{H,iso}(A,B)=\inf\limits_{T\in\mathcal{E}(N)}d_H(A,T(B)$$
$d_{H,iso}$ is sensitive to Shape + Size + Placement.

Gromov-Hausdorff Distance

How to compare shapes that do not have a common embedding?
For example, two abstract metric spaces $(X,d_X)$ and $(Y,d_Y)$. $$d_{GH}(X,Y)=\inf d_H(f(X),g(Y)).$$
