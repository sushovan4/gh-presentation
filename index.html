A shape can be modeled as a metric space $(X,d_X)$.

We need an appropriate notion of a distance measure $d_?(X,Y)$ so that


  1. $d_?(X,Y)$ defines a pseudo-metric on the class of metric spaces.
  2. $d_?(X,Y)$ large $\iff$ very different shape
  3. $d_?(X,Y)$ small $\iff$ $X=Y$ upto a class of deformation.

Hausdorff Distance


Nearest neighbor distance
Red points $\to$ green points

Let $A,B\subseteq\mathbb{R}^N$. The directed Hausdorff distance from $A$ to $B$ is defined by $$\overrightarrow{d}_H(A,B)=\sup_{a\in A}\inf_{b\in B}\|a-b\|.$$
For any two compact $A,B\subseteq\mathbb{R}^N$,
  1. $d_H(A,B)\geq0$,
  2. $d_H(A,B)=d_H(B,A)$,
  3. $d_H(A,B)=0\iff A=B$, and
  4. $d_H(A,C)\leq d_H(A,B)+d_H(B,C)$.


$d_H$ is sensitive to Shape + Size + Placement.

Let $\mathcal{E}(N)$ denote the class of Eulidean isometries or distance preserving maps $T:\mathbb{R}^N\to\mathbb{R}^N$ i.e. $\|T(a)-T(b)\|=\|a-b\|$.

  1. For $N=1$, $T$ is translation or reflection.

  2. For $N=2$, $T$ is rotation, translation or reflection.
For $X,Y\subseteq\mathbb{R}^N$, we define $$d_{H,iso}(X,Y)=\inf\limits_{T\in\mathcal{E}(N)}d_H(X,T(X))$$ $d_{H,iso}$ is sensitive to Shape + Size + Placement.

Gromov-Hausdorff Distance


How to compare shapes that do not have a common embedding?

Isometric Embedding

For example, two abstract metric spaces $(X,d_X)$ and $(Y,d_Y)$.

$$d_{GH}(X,Y)=\inf\limits_{f,g,Z} d^Z_H(f(X),g(Y)).$$

$d_{GH}$ vs $d_{H,iso}$

Let $X,Y\subseteq(\mathbb{R}^N,\|\cdot\|)$ be compact. Then, both $d_{GH}(X,Y)$ and $d_{H,iso}(X,Y)$ are defined. $$d_{GH}(X,Y)\leq d_{H,iso}(X,Y)\leq c_N\sqrt{Md_{GH}},$$ where $M=\max\{diam(X),diam(Y)\}$.
For $X,Y\subseteq\mathbb{R}^1$, we have $$d_{GH}(X,Y)\leq d_{H,iso}(X,Y)\leq\left(1+\frac{1}{4}\right)d_{GH},$$ and the upper bound is tight.

Computing $d_{GH}$

For $|X|,|Y|\leq k$, compuing $d_{GH}(X,Y)$ takes $O(2^k)$ time.
For $X,Y\subset\mathbb{R}^1$ and $|X|,|Y|\leq k$, compuing $d_{H,iso}(X,Y)$ takes $O(k\log{k})$ time.
